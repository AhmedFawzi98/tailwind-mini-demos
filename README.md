# Tailwind CSS Mini Components

A collection of small, focused UI components built using semantic HTML and Tailwind CSS.

Each demo represents a single component with an emphasis on layout, responsiveness, and clean utility usage.

## Goals
- Practice building UI components using utility-first CSS
- Improve layout intuition (flex, grid, sizing)
- Keep components responsive, reusable

## Tailwind Usage:
- Tailwind CSS is used via CLI (not via PostCSS as a plugin) for simplicity and due to no build tool usage
- general.css is precompiled into `output.css` 
- No build step is required to run the demos
